## Install (require node >=14.15.1 and yarn >=^1.22.5)
```bash
yarn
```
## Start in dev mod at http://localhost:3000/
```
yarn dev
```
## Start in production at http://localhost:3000/
```
yarn build
yarn start
```
## Run test (run on dev mode)
```
yarn ci
```
